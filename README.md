# HID_UTILL

## Prerequisites
```
1)pip3 install hid
2)Copy the 2 dll's that are in "x64" folder into:
C:\Windows\System32
```

## Installation
Your installtion folder should look like this:
```
.
├── install_folder
│   ├── HID_UTILL.py
│   ├── include_dll_path
│   │   └── __init__.py
│   ├── x64
│   │   └── hidapi.dll
│   └── x86
│       └── hidapi.dll
```
